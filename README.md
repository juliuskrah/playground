# Websocket Demo

This demo consists of the following projects

1. UI - [ui](/ui)
2. Gateway - [gateway](/gateway)
3. Backend - [micronaut](/micronaut)

## Running the UI

`mvnw clean spring-boot-run`

## Running the gateway

First download the dependencies `dotnet restore` then `dotnet run`

## Running the backend

`gradlew clean run`
